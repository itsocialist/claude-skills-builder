# Hire Designer Workflow

Use this workflow when contracting a graphic designer for logo or branding assets.

## 1. Define Requirements

Create a design brief covering:
- Brand values and personality
- Target audience description
- Deliverables list (logo, favicons, social cards)
- Technical requirements (formats, sizes)
- Reference/inspiration examples

## 2. Select Platform

**Recommended: 99designs** for logo contests
- Budget: $400-1200 depending on package
- Contest format provides multiple concepts
- Money-back guarantee

**Alternative: Dribbble** for direct hire
- Browse portfolios for specific style match
- Higher cost but more personalized

## 3. Launch Contest/Brief

// turbo
```
Open 99designs.com and navigate to Logo Design
```

- Submit detailed brief from Step 1
- Set appropriate budget tier
- Define timeline (typically 7 days for contest)

## 4. Review Submissions

- Rate all submissions using platform tools
- Provide feedback on promising concepts
- Request revisions from top 3-5 designers
- Narrow to finalist

## 5. Select Winner & Finalize

- Choose winning design
- Receive all file formats (AI, SVG, PNG)
- Verify files meet requirements:
  - [ ] Primary logo (horizontal)
  - [ ] Stacked logo
  - [ ] Icon/favicon (16, 32, 180, 512px)
  - [ ] Dark and light variants
  - [ ] SVG source files

## 6. Implementation

Handoff to frontend team to implement:
- Navigation header
- Favicon
- Open Graph images
- Footer/branding elements
