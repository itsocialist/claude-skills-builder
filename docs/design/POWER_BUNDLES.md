# Power Bundles: Product Concept

**Status**: Draft
**Related Issue**: #37
**Strategy**: Sell outcomes as packages, not tools as features.

---

## Bundle 1: Research Studio
> "From question to polished brief in minutes."

**The Cake**: A 5-10 page research report with citations, executive summary, and appendix.

| Skill | Role |
|-------|------|
| Web Search | Gathers sources from the web |
| Document Summarizer | Distills long sources into key points |
| Citation Formatter | Generates proper bibliography |
| Report Builder | Structures and formats the final PDF |

**Example Prompt**: "Research the competitive landscape for AI writing assistants in 2024."

---

## Bundle 2: Content Engine
> "From idea to publish-ready content suite."

**The Cake**: A blog post, 3 social snippets, an SEO meta description, and a hero image.

| Skill | Role |
|-------|------|
| Blog Writer | Drafts long-form content |
| SEO Optimizer | Adds meta tags, keywords |
| Social Media Snippeter | Creates platform-specific posts |
| Image Generator | Creates a featured image |

**Example Prompt**: "Write a blog post about the benefits of AI in healthcare."

---

## Bundle 3: Data Analyst
> "From raw data to executive dashboard."

**The Cake**: A formatted spreadsheet with charts, summary statistics, and insights.

| Skill | Role |
|-------|------|
| CSV Parser | Ingests and cleans data |
| Formula Builder | Creates calculation columns |
| Chart Generator | Visualizes key metrics |
| Insight Writer | Summarizes findings in plain English |

**Example Prompt**: "Analyze this sales data and create a Q3 performance dashboard."

---

## Bundle 4: Project Kickstart
> "From vague idea to actionable plan."

**The Cake**: A project brief, timeline spreadsheet, and risk register.

| Skill | Role |
|-------|------|
| Requirements Gatherer | Extracts scope from description |
| Timeline Builder | Creates phased milestones |
| Task Breakdown | Generates work items |
| Risk Analyzer | Identifies potential blockers |

**Example Prompt**: "Plan a 3-month product launch for a B2B SaaS tool."
